We are currently studying Geomatics Engineering at Pashchimanchal Campus. We created this website as our final year project. This website contain the map of Kaskikot ward 24,Pokhara.
The main aim of creating this website is to help the user to search for the nearest critical facility like health facility ,community building, open space, school, financial institution inside the ward and show the shortest route to these facility.
This website can be used to search various geographic attributes such as school, hospital or health facility, community building, financial institution.Also we can visualiza the road network and waterways like stream and river located in the ward. Thank you for visiting our website and we hope you find it informative and engaging. Feel free to contact us if you have any questions or feedback.
